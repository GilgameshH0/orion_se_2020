Задание по JDBC

 

Создать структуру данных, описывающих соревнования между командами по футболу. 

 

Есть сущности:
1. Игрок
- Имя игрока
- Команда, в которой игрок играет

 

2. Команда
- Имя команды 
- Ссылка на сайт команды

 


3. Чемпионат - несколько матчей 
- Название чемпионата.

 


4. Матч - описывает встречу между командами
- Первая команда участница
- Вторая команда участница 
- Результат: Победали команда 1, победила команда 2, ничья. 

 

5. Период - период конкретного матча.
- Название периода (первый тайм, второй тайм, первый дополнительный тайм, второй дополнительный тайм, дполнительных таймов может не быть)
- Основное время (для обычныйх таймов всегда 45 минут)
- Количество добавленного времени (может быть различное для каждого тайма) 
- Матч,  к которому относится период. 

 


6. Голы:
- Кто забил гол
- Время гола
- Период, в котором забили гол. 

 

 

Считаем, что игроки между командами не переходят.
Задания:
1. Создать необходимые таблицы для хранения информации. Помни про внешние ключи и огранчения целостности. 
2. Заполнить таблицы таким образом, чтобы было несколько команд с игроками (не обязательно 11), было проведено несколько чемпионатов с матчами и кто-то забивал голы.

 

Провести аналитику:
1. Найти игрока, который забил больше всего голов во всех чемпионатах
2. Найти игрока, который забил больше всего голов в одном матче
3. Найти игрока, который забил больше всего голов в одном чепионате
4. Найти матч, с самым большим количеством забитых голов (счиаются голы  обеих команд)
5. Найти самый продолжительный матч (считаются все таймы с дополнительным временем) 
6. Найти команду, игроки которой забили меньше всего голов.
7. Вывести статистику по чемпионатам в формате:название чемпионата, сколько голов было забито
8. Вывести только те чемпионаты , в которых было забито не меньше 10 голов.
